# 🧠 Customer Segmentation using Mall Data

## 📌 Overview
This project focuses on **Customer Segmentation** using a mall dataset to classify customers based on their spending behavior. The goal is to identify patterns and group customers into meaningful categories such as high and low spenders.

## ⚙️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- scikit-learn

## 📂 Dataset
The dataset used is **Mall_Customers.csv**, which includes:
- Age
- Annual Income (k$)
- Spending Score (1–100)

## 🚀 Implementation
- Data preprocessing and exploration using Pandas  
- Feature selection (Age, Income, Spending Score)  
- Created labels for segmentation (High vs Low spending)  
- Applied Decision Tree Classifier for segmentation  
- Visualized decision boundaries using tree plots  

## 📈 Model Details
- Algorithm: Decision Tree Classifier  
- Depth: 3  
- Target: Spending behavior (High / Low)  

## 📊 Output
- Classification of customers into segments  
- Visualization of decision tree for interpretability  
- Predictions based on input features  

## ▶️ How to Run
```bash
pip install numpy pandas matplotlib scikit-learn
python decision_tree-mall.py
```

## 🧠 Key Learnings
- Basics of customer segmentation  
- Classification using Decision Trees  
- Feature engineering and labeling  
- Model visualization and interpretation  

## 📌 Conclusion
This project demonstrates how machine learning can be used to segment customers and support data-driven business decisions in retail.
